The Dataset

Synthetic car dataset with 600 samples across 15 brands. Features include brand, year, showroom price, mileage, fuel type, transmission, seller type, and ownership history.

⚙️

How It Works

We train 5 regression models (Linear, Ridge, Random Forest, Gradient Boosting, SVR), evaluate with R², MAE, RMSE, and 5-fold cross-validation to pick the best.

🎯

Key Concepts

Regression predicts continuous values (price) from features. We encode categorical variables, scale features, and measure error to evaluate model quality.
