The Dataset

The Iris dataset was introduced by Ronald Fisher in 1936. It contains 150 samples from three species: Setosa, Versicolor, and Virginica. Each sample has four features measured in centimeters.

⚙️

How It Works

We train 5 classifiers (Random Forest, SVM, KNN, Logistic Regression, Gradient Boosting), evaluate each on an 80/20 split with 5-fold cross-validation, and pick the best performer.

🎯

Key Concepts

Classification is supervised learning where a model predicts categories from labeled data. We measure quality with accuracy, precision, recall, F1-score, and confusion matrices.
